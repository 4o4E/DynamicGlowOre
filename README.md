# GrowOre
Minecraft 动态发光矿石 材质包

# 使用方法
1. 下载对应版本的材质包
2. 将材质包放进 游戏文件夹/.minecraft/resourcepacks 文件夹
3. 在游戏中 选项 -> 资源包 中选中本材质(优先级需要比其他修改了矿石贴图的材质包高)
